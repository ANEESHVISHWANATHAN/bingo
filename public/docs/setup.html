üîê Authentication System

Required Endpoints

POST /api/signup

Purpose: Create new user account

Request Body:

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "securepassword",
  "role": "buyer" // or "seller"
}

Response:

{
  "token": "jwt_token_here",
  "user": {
    "id": "user_id",
    "email": "john@example.com",
    "role": "buyer"
  }
}
POST /api/login

Purpose: Authenticate existing user

Request Body:

{
  "email": "john@example.com",
  "password": "securepassword"
}

Response: Same as signup

Implementation Steps

  1. Install dependencies: npm install bcrypt jsonwebtoken
  2. Hash passwords before storing (use bcrypt with salt rounds ‚â• 10)
  3. Generate JWT tokens with 24-hour expiration
  4. Store token in httpOnly cookies or localStorage
  5. Create authentication middleware to protect routes
  6. Implement role-based access control (buyer, seller, admin)
Security: Never store passwords in plain text. Always hash with bcrypt or argon2. Use HTTPS in production.

Image Upload

Recommended approach using cloud storage:

  1. Use Cloudinary, AWS S3, or similar service
  2. Install SDK: npm install cloudinary
  3. Create upload endpoint: POST /api/upload
  4. Return image URL to store in product data
// Example Cloudinary setup
import { v2 as cloudinary } from 'cloudinary';

cloudinary.config({
  cloud_name: process.env.CLOUDINARY_CLOUD_NAME,
  api_key: process.env.CLOUDINARY_API_KEY,
  api_secret: process.env.CLOUDINARY_API_SECRET
});

// Upload endpoint
app.post('/api/upload', upload.single('image'), async (req, res) => {
  const result = await cloudinary.uploader.upload(req.file.path);
  res.json({ url: result.secure_url });
});

üí≥ Payment Gateway Integration

Stripe Integration

Install: npm install stripe

  • STRIPE_SECRET_KEY
  • STRIPE_WEBHOOK_SECRET

Endpoint:

POST /api/create-stripe-session

PayPal Integration

Install: npm install @paypal/checkout-server-sdk

Environment Variables:

  • PAYPAL_CLIENT_ID
  • PAYPAL_SECRET

Endpoint:

POST /api/create-paypal-order

Stripe Implementation Example

import Stripe from 'stripe';
const stripe = new Stripe(process.env.STRIPE_SECRET_KEY);

app.post('/api/create-stripe-session', async (req, res) => {
  const { items } = req.body;
  
  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: items.map(item => ({
      price_data: {
        currency: 'usd',
        product_data: { name: item.name },
        unit_amount: item.price * 100,
      },
      quantity: item.quantity,
    })),
    mode: 'payment',
    success_url: `${req.headers.origin}/success`,
    cancel_url: `${req.headers.origin}/checkout`,
  });
  
  res.json({ sessionId: session.id });
});
